# MechTech - Mechanical Tools Store

This project is a fully functional web application for browsing and purchasing mechanical tools. It provides a simple and responsive interface where users can explore product details, view images, read reviews, and access technical information. The website is built with HTML, CSS, and JavaScript, and is designed to offer a clean and user friendly shopping experience.

## Features

### Product Catalogue
The homepage and supporting pages display a variety of mechanical tools with images and descriptions. Each product has an individual section that highlights specifications, use cases, and visual references.

### Product Detail Pages
Files such as tech.html and reviews.html provide additional product information including:
- Technical specifications
- Customer reviews
- Product demonstration images

### User Interaction
The site includes:
- A signup page for new users
- A login page for returning users
- A structured navigation flow integrated through header elements

### Image Gallery
Multiple product images are included to help users visually understand the tools. The repository contains support images such as:
- Tech.jpg
- Mechanical_tools.jpg
- Mechanical_tools2.jpg
- Various numbered product images

### Responsive Styling
The website uses styles.css to ensure compatibility across different screen sizes. The layout adjusts for desktop and mobile viewing.

### JavaScript Functionality
The app.js file handles dynamic behavior across pages to improve user experience.

## Project Structure

## How to Run

1. Download or clone the repository.
2. Open index.html in any modern web browser.
3. Navigate through the website using the menu and links.

No server setup is required since the project runs entirely on the frontend.

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project demonstrates a simple and functional ecommerce style interface designed for showcasing and selling mechanical tools. It focuses on presentation, clarity, and easy navigation without involving back end systems.
